# KANA SWIM CLUB β

## GitHubへのアップロード

1. このZIPを解凍します。
2. `ksc_site` フォルダの中身をすべてGitHubのアップロード画面へドラッグします。
   - `index.html`
   - `style.css`
   - `script.js`
   - `staff.json`
   - `assets` フォルダ
3. GitHub画面下部の `Commit changes` を押します。
4. GitHub Pages / Cloudflare Pagesで公開できます。

## 注意
ZIPファイル自体ではなく、解凍した中身をアップロードしてください。
